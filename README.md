# F6S (f6s)

F6S (https://www.f6s.com/) is the world's largest startup and founder community platform, connecting over 1.6 million tech startups and 2.9 million entrepreneurs with accelerator programs, grants, funding, jobs, and deals. Founders use F6S to discover and apply to accelerator and grant programs, find investors, recruit talent, and access discounted tools and services. Organizations including corporations, governments, universities, and investors use F6S to run application management workflows, source deal flow, and engage with the global startup ecosystem. The platform processes over 700,000 applications annually and delivers approximately €3 billion to startups each year. F6S also provides a developer API at developers.f6s.com for integrating with its application management, program discovery, and community data capabilities.

## APIs

### F6S API

The F6S API provides programmatic access to the F6S platform, the world's largest startup and founder community. It enables developers to integrate with F6S's core capabilities including application management, program discovery, startup profiles, funding opportunities, and community data. The API supports authentication via OAuth and exposes endpoints for managing accelerator and grant program applications, querying startup and founder profiles, and accessing deals and job listings across the global F6S network of over 1.6 million tech startups and 2.9 million entrepreneurs.
